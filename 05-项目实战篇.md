## 项目一：RAG 知识问答系统（2-3周）

### 项目描述
基于企业文档构建智能知识问答系统，支持多格式文档解析、混合检索、多轮对话。

### 技术栈
- 框架：LangChain + LangGraph
- 向量数据库：Milvus
- Embedding：BGE-M3
- Re-ranker：BGE-Reranker-v2
- LLM：Qwen2.5-72B / DeepSeek-V3
- 前端：Streamlit / Gradio
- 后端：FastAPI

### 核心架构
```
用户查询 → 查询理解（改写/分类）
         → 混合检索（向量+BM25+RRF融合）
         → Re-ranking（交叉编码器重排序）
         → 上下文组装（压缩+排序）
         → LLM 生成（带引用标注）
         → 后处理（幻觉检测+格式化）
```

### 实现步骤
1. 文档处理流水线（PDF/Word/Markdown/HTML）
2. 检索系统（稠密+稀疏+混合+重排序）
3. 生成系统（System Prompt + 引用溯源 + 幻觉检测）
4. 多轮对话（历史管理 + 指代消解）
5. 评估优化（RAGAs + 测试集）

### 项目亮点
- 混合检索 + Re-ranking 使 Recall@5 提升 15%
- Semantic Chunking 使准确率提升 10%
- 幻觉检测将无据回答率从 12% 降到 3%

## 项目二：多 Agent 协作系统（2-3周）

### 项目描述
用 LangGraph 实现 Supervisor 模式的多 Agent 研究助手。

### 系统架构
```
用户需求 → Supervisor Agent（规划/调度）
           ├→ Researcher Agent（搜索/检索）
           ├→ Analyst Agent（分析/推理）
           ├→ Writer Agent（撰写报告）
           └→ Reviewer Agent（审核/修改）
```

### 技术栈
- 核心框架：LangGraph
- LLM：GPT-4o / Claude 3.5 / Qwen-Max
- 工具：Tavily Search、Python REPL、文件读写
- 持久化：SQLite + LangGraph Checkpointing

### 实现步骤
1. 定义 Agent 角色（Supervisor + 4 个 Worker）
2. LangGraph 实现（State + 条件边 + 循环）
3. 工具集成（搜索 + 代码执行 + 文件操作）
4. Human-in-the-loop（关键决策审核）
5. 状态持久化（Checkpointing + 中断恢复）

### 项目亮点
- Supervisor 任务分解和动态调度
- Human-in-the-loop 关键决策审核
- Checkpointing 支持中断恢复

## 项目三：生产级 Agent 应用（2-3周）

### 项目描述
构建生产级个人助理 Agent，支持工具调用、长期记忆、安全防护。

### 技术栈
- 框架：LangGraph + FastAPI
- LLM：多模型路由（小模型+大模型）
- 记忆：Redis（短期）+ PostgreSQL + pgvector（长期）
- 安全：NeMo Guardrails
- 监控：LangSmith + Prometheus
- 部署：Docker + Kubernetes

### 核心模块
1. **智能路由**：意图识别 + 模型选择 + 工具选择
2. **工具调用系统**：MCP 协议集成 + 注册发现
3. **记忆系统**：短期 + 长期 + 实体记忆
4. **安全防护**：输入检测 + 输出过滤 + 权限控制
5. **可观测性**：全链路追踪 + 监控告警

### 项目亮点
- 多模型路由，简单任务成本降低 70%
- 完整安全防护体系
- 长期记忆实现个性化服务
- Docker 化一键部署
